Objetos

La estructura clave del lenguaje

Casi todo en JavaScript es un objeto o se comporta como uno. Los objetos son colecciones de pares clave-valor que modelan entidades del mundo real y son la base de la programación estructurada del lenguaje.

Aprenderás a crearlos con literales, a manipular sus propiedades, a desestructurarlos para extraer datos de forma elegante y a copiarlos/combinarlos con el operador spread.

Conceptos que verás

📦

Objetos literales

Crear objetos con { clave: valor }, acceder y modificar propiedades con notación de punto o corchetes.

🔍

Desestructuración

Extraer múltiples propiedades en una sola línea con const { nombre, edad } = usuario y renombrado con :.

🛠️

Métodos de objetos

Object.keys(), Object.values(), Object.entries(), Object.assign() y Object.freeze() para trabajar con ellos.

📋

Spread operator

{ ...otro } para copiar, fusionar y clonar objetos de forma inmutable, sin perder referencia.

🔗

Referencia vs valor

Los objetos se asignan por referencia: dos variables pueden apuntar al mismo objeto. Aprende a evitar mutaciones inesperadas.

📚

JSON

Serialización con JSON.stringify() y parseo con JSON.parse() para comunicarte con APIs.
