Arreglos

Listas, colecciones y transformaciones

Los arreglos (arrays) son la estructura de datos más usada en JavaScript: listas ordenadas de elementos accesibles por índice. Permiten almacenar colecciones heterogéneas y aplicar transformaciones funcionales muy expresivas.

Conocerás los métodos funcionales como map, filter, reduce, forEach y find que son el pan de cada día en el desarrollo moderno.

Conceptos que verás

📚

Arreglos básicos

Crear arrays con [], acceder por índice, la propiedad length y los métodos push, pop, shift, unshift.

🔁

Iteración

Recorrer un array con for, for...of y forEach — elegir la forma correcta para cada caso.

🧪

Métodos funcionales

map para transformar, filter para filtrar, reduce para acumular, find y some para buscar.

🧊

Inmutabilidad

concat, slice y el operador spread para crear nuevos arrays sin mutar el original.

🔎

Búsqueda y orden

indexOf, includes, findIndex, sort y reverse para localizar y reordenar elementos.

🧬

Arrays multidimensionales

Matrices y acceso anidado, recorridos con bucles anidados y casos prácticos (tableros, grids).
