Estructuras de Control

El flujo de tu programa

Las estructuras de control permiten tomar decisiones y repetir tareas. Son el pilar de cualquier programa no trivial: condicionales para ramificar el flujo y bucles para iterar sobre colecciones o repetir acciones.

Desde el clásico if / else hasta el switch, pasando por for, while y los métodos forEach y map que son la forma moderna de recorrer arrays.

Conceptos que verás

🔀

if / else / else if

Condicionales clásicos, evaluación de expresiones booleanas, operadores lógicos &&, ||, ! y anidación de condiciones.

🧭

switch

Selección múltiple basada en casos, break, comportamiento de fallthrough y cuándo conviene sobre if / else.

🔁

for / for...of / for...in

Bucles clásicos con contador, recorrido de iterables con for...of y enumeración de propiedades con for...in.

⏳

while / do...while

Bucles condicionados, cuándo usarlos en lugar de for y el peligro de los bucles infinitos.

🔂

forEach

Iteración funcional de arrays: arr.forEach((item, i) => ...) y comparación con for...of.

🗺️

map

Transformar un array en otro: arr.map(x => x * 2), composición y retorno implícito en arrow functions.
