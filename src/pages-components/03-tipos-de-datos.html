03

Tipos de Datos

Los valores que manipula tu código

JavaScript es un lenguaje de tipado dinámico: las variables no tienen un tipo fijo y pueden cambiar de un tipo a otro durante la ejecución. Conocer los tipos primitivos y de referencia es esencial para evitar bugs sutiles.

En esta sección exploramos los primitivos (string, number, boolean, null, undefined, symbol, bigint) y el objeto global Math, junto con operadores, precedencia y template strings.

Conceptos que verás

🔤

Strings

Cadenas de texto, concatenación, métodos como slice, includes, replace y propiedades como length.

🔢

Números y operadores

Tipo number (entero y decimal), operadores aritméticos, de comparación y lógicos. Cuidado con la precisión decimal.

🧮

Objeto Math

Constantes y métodos matemáticos: Math.PI, Math.random(), Math.round(), Math.max(), Math.min(), etc.

⚖️

Orden de operaciones

Precedencia y asociatividad: cómo JavaScript evalúa expresiones complejas y por qué los paréntesis son tus mejores amigos.

📐

Template Strings

Literales con `backticks`, interpolación ${variable} y strings multilínea para HTML dinámico.

✅

Booleanos

Valores true / false, valores falsy/truthy y la base para las estructuras de control que verás más adelante.
