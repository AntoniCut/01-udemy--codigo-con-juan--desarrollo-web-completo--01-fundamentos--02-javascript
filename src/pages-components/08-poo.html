Programación Orientada a Objetos

Modela el mundo con objetos

La Programación Orientada a Objetos (POO) en JavaScript se basa en prototipos. Aunque class ofrece una sintaxis familiar, el motor sigue usando prototipos internamente — entender ambos modelos es la diferencia entre usar el lenguaje y dominarlo.

Aprenderás a trabajar con this, a crear objetos con constructores y prototype, y a usar la sintaxis moderna de clases con herencia mediante extends.

Conceptos que verás

👉

this

El contexto de ejecución: cómo se enlaza this según cómo se llame la función (método, función, constructor, evento).

🧬

POO con prototipos

La cadena de prototipos, Object.create(), __proto__ y cómo JavaScript implementa la herencia de forma nativa.

🛠️

Constructores

Funciones constructoras con new, métodos en prototype y métodos estáticos.

🏛️

Clases (ES6)

Sintaxis class, constructor(), métodos de instancia, get / set y campos privados (#).

🌳

Herencia

extends, super(), sobrescritura de métodos y polimorfismo en JavaScript.

🧪

Buenas prácticas

Cuándo usar class vs funciones, composición frente a herencia, y patrones modernos de POO.
