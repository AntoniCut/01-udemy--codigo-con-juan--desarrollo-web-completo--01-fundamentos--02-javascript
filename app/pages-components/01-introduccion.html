Introducción

Primeros pasos en JavaScript

JavaScript es el lenguaje de programación que da vida a las páginas web. Nacido en 1995 como un simple script para validar formularios en el navegador, se ha convertido en la tecnología más usada del mundo: frontend, backend, mobile, desktop, IoT y mucho más.

En esta sección introductoria aprenderás los fundamentos esenciales: cómo incluir JavaScript en una página, cómo se ejecuta y cómo empezar a escribir tus primeras líneas de código entendiendo la consola del navegador.

Conceptos que verás

📜

Historia de JavaScript

Orígenes en Netscape, evolución a ECMAScript y el salto a Node.js que llevó JavaScript del navegador al servidor.

🌐

¿Dónde se ejecuta?

En el navegador mediante el motor V8 (Chrome, Edge, Node) o SpiderMonkey (Firefox). Conoce el entorno de ejecución y el rol del DOM.

🛠️

Herramientas de desarrollo

La consola del navegador (DevTools), el inspector y console.log como tu mejor aliado para depurar.

📝

Tu primer script

El clásico Hola Mundo, cómo incluir JavaScript en HTML con la etiqueta <script> y atributos defer / async.

💬

Comentarios y sintaxis

Comentarios de una línea (//) y de bloque (/* */), punto y coma, y las reglas básicas de estilo del lenguaje.

🚀

¿Qué viene después?

Una panorámica del temario: variables, tipos, objetos, funciones, POO y asincronía — todo lo necesario para dominar los fundamentos.
